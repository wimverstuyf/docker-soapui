Docker SoapUI

If you are seeing this page, this means your docker container is successfully running!

You are ready to start using Docker SoapUI.



To Use...

Send a `curl` request to this URL with the following code 
    $ curl --form "project=@/path/to/soapui-project.xml" \
           --form "suite=TestSuite" \
           --form "properties=@pathToGlobalPropertiesFile" \
           --form "option=-r" \
           this URL
Use a REST client to POST to with these variables
  • project - the actual content of the SoapUI project xml (Required).
  • suite - the suite to run (usually will be TestSuite if left unchanged from SoapUI) (Optional).
  • properties - File containing global properties. Exported from SOAPui preferences editor (Optional)
  • option - Text or file containing Testrunner options, e.g. "-r" for short summary. Can point to a file with lines of options (Optional).
